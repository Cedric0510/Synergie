# SCARD - Propositions de cartes fusionnees (prototype)

But
- Liste des cartes fusionnees proposees (4 paliers : blanc/bleu/jaune/rouge).
- Le joueur peut choisir n'importe quel palier jusqu'a sa couleur actuelle.
- Un seul bloc d'effet par carte (pas de cout de lancement separe).

Notes
- Propositions a retravailler.
- Les cartes sources viennent de cards.json (ids listes).

Rituels (cartes a jouer)

1) Deshabillage (theme : retrait de vetements)
   Sources : white_001, blue_001, (inspiration : white_010)
   - Blanc : la cible enleve 1 vetement.
   - Bleu : la cible enleve 2 vetements.
   - Jaune : striptease / enleve jusqu'aux sous-vetements.
   - Rouge : nudite totale.

2) Confidences (theme : mots / intimite)
   Sources : white_004, blue_003, yellow_004, blue_004
   - Blanc : compliment sincere.
   - Bleu : chuchoter des mots coquins a l'oreille.
   - Jaune : partager un fantasme.
   - Rouge : reveler un secret tres intime.

3) Caresse / Massage (theme : toucher)
   Sources : blue_005, blue_006, yellow_003, red_005
   - Blanc : caresse legere (bras/nuque/dos).
   - Bleu : massage 1 min (zone au choix).
   - Jaune : caresse intime (cachee).
   - Rouge : caresse intime (exposee).

4) Baisers (theme : bisous)
   Sources : inspiration blue_013, yellow_012
   - Blanc : bisous sur la joue.
   - Bleu : bisous dans le cou.
   - Jaune : baiser langoureux sur la bouche.
   - Rouge : baisers intimes.

5) Provocation / Show (theme : display sexy)
   Sources : white_005, yellow_002 (danse), yellow_006
   - Blanc : pose sexy (courte).
   - Bleu : petit show sensuel (court).
   - Jaune : danse sensuelle 30 sec.
   - Rouge : show erotique (plus intense).

6) Stimulation ciblee (theme : teasing / pincements)
   Sources : yellow_001, red_006
   - Blanc : teasing leger (zones non intimes).
   - Bleu : pincements legers.
   - Jaune : stimulation plus appuyee.
   - Rouge : frottement intime.

7) Plaisir guide (theme : donner du plaisir)
   Sources : red_001, red_002, red_003, red_007
   - Blanc : plaisir leger (zones non intimes).
   - Bleu : plaisir avec les mains.
   - Jaune : plaisir oral.
   - Rouge : sextoy ou point G (choix).

8) Sextoy (theme : jouets)
   Sources : yellow_005, red_004, red_003
   - Blanc : choisir un sextoy (ou mimer si aucun).
   - Bleu : sextoy soft (court).
   - Jaune : sextoy cache.
   - Rouge : sextoy visible / impose.

9) Desenchantement (theme : retirer un enchantement)
   Sources : white_003, blue_004, yellow_002
   - Blanc : detruire 1 enchantement.
   - Bleu : detruire 1 + revelation.
   - Jaune : detruire 1 + bonus pioche.
   - Rouge : detruire 2 OU detruire 1 protege.

10) Pioche / Avantage (theme : draw)
    Sources : white_002, blue_002, yellow_002
    - Blanc : pioche 1-2 cartes.
    - Bleu : pioche 2 cartes.
    - Jaune : pioche 3 cartes.
    - Rouge : pioche 4 cartes OU pioche + gain PI.

Enchantements (effets persistants)

11) Bisous (enchantement)
    Sources : blue_013, yellow_012
    - Blanc : bisou joue a chaque tour.
    - Bleu : bisou cou a chaque tour.
    - Jaune : bisou bouche a chaque tour.
    - Rouge : bisou intime a chaque tour.

12) Massage (enchantement)
    Sources : blue_011, blue_012
    - Blanc : massage tete a chaque tour.
    - Bleu : massage epaules a chaque tour.
    - Jaune : massage dos a chaque tour.
    - Rouge : massage intime a chaque tour.

13) Controle / Pression (enchantement)
    Sources : yellow_011, red_013, red_015, red_014
    - Blanc : contre leger / contrainte mineure.
    - Bleu : contrainte d'action (faire X pour jouer).
    - Jaune : pression PI a chaque tour.
    - Rouge : obligation totale / refus impossible.

14) Tenue / Rhabillage (enchantement)
    Sources : white_009, white_010, yellow_013
    - Blanc : chastete (pas de retrait de sous-vetements).
    - Bleu : pression striptease (retirer un vetement par tour).
    - Jaune : rhabillage auto 1 vetement par tour.
    - Rouge : verrouillage d'etat (regle a definir).

Special / a conserver uniques
- Ultima (red_016) reste unique.
- Negociations (green_001..008) restent uniques ; ne contrent pas les effets rouges.
